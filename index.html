O que são Tags?

(Voltar)

Podemos começar falando, primeiramente, sobre o que são tags:

- As Tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.

Dentre essas, podemos analisar tipos diferentes de Tags, como Tags texto, que transformam textos, de Link, e até para detalhes.

Tags que transformam o texto

(Voltar)

De muitas Tags que podem ser utilizadas para transformar um texto, vou citar 10, de forma resumida:

  • Mark - Marca texto
  • Small - Deixa texto pequeno
  • Strong - Deixa o texto em Negrito (pode ser substituído por b)
  • Sub - Deixa o texto "elevado"
  • Sup - Deixa o texto "baixo"
  • Font - Muda o tamanho, fonte e cor do texto (Não suportado em HTML5)
  • Abbr - Define o texto como sigla, identificado pelo atributo title="", dentro do abbr, que altera todas as siglas por CSS usando o id do title
  • Del - Deixa o texto rasurado
  • i - Deixa o texto em Itálico
  • u - Deixa o texto sublinhado

Tag Link

(Voltar)

Das muitas que existem, a tag a é o único Hyperlink que eu já estudei até agora, no dia que estou fazendo o primeiro Desafio da DIO.

  • A Tag a é chamada de âncora, ou seja, ela, através de um texto, quando clicado, o usuário é direcionado para um outro site, abrindo na mesma aba (com o atributo target="_self") dentro do a ou em outra (com o atributo target="_blank")

    • - Dentro da Tag, é usado o atributo href="link", onde é armazenado o link de destino, como por exemplo:

    Quando clicar aqui, você será direcionado para o site do Youtube.

    - Ou, podemos usar o href=#id, para direcionarmos o usuário para algum lugar da página atual, com o identificador: id, utilizado após o #.

    Quando clicar aqui, você voltará para o início desta página.
